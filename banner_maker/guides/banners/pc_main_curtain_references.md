## 발견된 패턴

### 1. 텍스트 구성 (Composition)

3가지 구성 패턴이 발견되었습니다.

| 구성 타입 | 빈도 | 설명 |
|---|---|---|
| **서브+메인 가로 1줄** | 3/5 (최다) | 서브(좌, Medium) + 메인(우, Bold) 가로 나란히 |
| **서브+메인 세로 2줄** | 1/5 | 서브(위, 40px) + 메인(아래, 72px) 세로 배치 |
| **메인 단독 1줄** | 1/5 | 메인 카피 하나만 중앙 배치 |

### 2. 폰트 사이즈 규칙

| 조건 | 메인 사이즈 | 서브 사이즈 |
|---|---|---|
| 세로 2줄 | 72px | 40px |
| 단독 1줄 | 72px | — |
| 가로 1줄, 합계 ≤ 24자 | 72px | 72px |
| 가로 1줄, 합계 > 24자 | 64px | 64px |

핵심: 가로 배치에서 **텍스트 합산 길이가 24자를 넘으면 64px로 축소**하여 공간을 확보합니다.

### 3. 폰트 웨이트 규칙

| 역할 | 웨이트 | 비고 |
|---|---|---|
| 메인 카피 | **Bold** | 예외 없음 |
| 서브 카피 | **Medium** | 예외 없음 |

### 4. 자간 (Letter Spacing)

**-2% 고정** (가이드 기준값, 변동 없음)

### 5. 폰트 패밀리

**Gmarket Sans 고정** (원본 사례 중 Paperlogy 1건 있었으나 고정 정책에 따라 무시)

### 6. 텍스트 컬러

| 패턴 | 설명 |
|---|---|
| 기본 | 서브/메인 모두 흰색(#ffffff) |
| 강조 | 메인만 강조색(#fff58d 노란색) 또는 그라디언트 |

### 7. 배경색

5가지 배경색이 사용됨: 파랑, 연보라, 오렌지, 다크그레이, 빨강. 모든 사례에서 이미지도 함께 사용.

## 자동 결정 규칙 (Auto-Decision Rules)

시스템이 사용자 입력(메인카피, 서브카피)을 받았을 때 자동으로 적용할 규칙:

### Step 1: 구성 결정
```
서브카피 없음 → 메인 단독 1줄
서브카피 있음 + 서브 ≤ 12자 + 메인 ≤ 15자 → 세로 2줄
서브카피 있음 + 합계 ≤ 24자 → 가로 1줄
서브카피 있음 + 합계 > 24자 → 가로 1줄 (축소)
```

### Step 2: 폰트 사이즈 결정
```
세로 2줄 → 메인 72px / 서브 40px
단독 1줄 → 72px
가로 1줄 + 합계 ≤ 24자 → 둘 다 72px
가로 1줄 + 합계 > 24자 → 둘 다 64px
```

### Step 3: 웨이트/자간 결정
```
메인 → Bold / 자간 -2% (고정)
서브 → Medium / 자간 -2% (고정)
```

### Step 4: 컬러 결정
```
기본 → 모두 #ffffff
강조 모드 → 메인 #fff58d, 서브 #ffffff
```
