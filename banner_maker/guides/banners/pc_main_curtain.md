|---|---|---|

| 서브+메인 세로 2줄 | 1/5 | 서브(위, 40px) + 메인(아래, 72px) 세로 배치 |
| 메인 단독 1줄 | 1/5 | 메인 카피 하나만 중앙 배치 |

### 텍스트 정렬 방식

기준점: **배너 중앙 (width/2, height/2)**에서 시작하여 좌우/상하로 뻗어나가는 구조

| 구성 | 정렬 방식 |
|---|---|
| 메인 단독 1줄 | 중앙 기준, 메인 카피가 좌우 균등 배치 |
| 서브+메인 가로 1줄 | 중앙 기준, 서브(좌로) + 메인(우로) 뻗어나감 |
| 서브+메인 세로 2줄 | 중앙 기준, 서브(위로) + 메인(아래로) 뻗어나감 |

### 자동 결정 규칙

**Step 1: 구성 결정**
```
서브카피 없음 → 메인 단독 1줄
서브카피 있음 + 서브 ≤ 12자 + 메인 ≤ 15자 → 세로 2줄
서브카피 있음 + 합계 ≤ 24자 → 가로 1줄
서브카피 있음 + 합계 > 24자 → 가로 1줄 (축소)
```

**Step 2: 폰트 사이즈 결정**
```
세로 2줄 → 메인 72px / 서브 40px
단독 1줄 → 72px
가로 1줄 + 합계 ≤ 24자 → 둘 다 72px
가로 1줄 + 합계 > 24자 → 둘 다 64px
```

**Step 3: 웨이트 결정**
```
메인 → Bold (고정)
서브 → Medium (고정)
```
