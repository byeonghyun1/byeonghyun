## 모바일 배너

### 9. M 앱 스플래시 배너 (`m_app_splash`)

| 항목 | 값 |
|---|---|
| 위치 | APP > 스플래시 |
| 사이즈 | 1440 x 976 |
| 최대 파일 크기 | 300KB |
| 파일 형식 | png |
| 투명 배경 | 필수 |
| 라운딩 | 불가 |
| 테두리 | 불가 |

**여백:** 좌 144px / 우 144px

**로고 영역:** 1440 x 240, 중앙 배치, 최소 높이 120px (알바몬 로고 고정)

**텍스트 영역:** 1440 x 240, 상단 25% 내 중앙 배치, 메인 최대 16자, 서브 최대 18자

**오브젝트 영역:** 최소 너비 880px, 텍스트와 88px 간격

**폰트:**
- 메인: Pretendard Bold 88px (자간 -0.5%, 색상 #171717)
- 서브: Pretendard Regular 64px (자간 0%, 색상 #171717)

**레이아웃 케이스:**
- 로고 + 오브젝트 (로고 최대 높이 120px)
- 문구 + 오브젝트 (간격 88px)
- 문구 + 로고 + 오브젝트 (로고 최대 높이 72px, 간격 88px)

**제작 규칙:**
- 투명 배경 사용 필수
- 화면 중앙 로고 영역은 알바몬 로고 이미지 고정
- 광고 브랜드 로고&문구는 상단 25%내 중앙배치
- 좌우 최소 여백 144px 준수
- 브랜드 식별에 방해되는 오브젝트 사용 금지
- 인물 또는 캐릭터가 포함된 경우 주요부위가 잘리지 않도록 유의
- 문구영역과 오브젝트 여백 88px
