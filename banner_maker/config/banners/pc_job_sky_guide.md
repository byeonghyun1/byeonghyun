## 7. 이미지 영역 (image_box)

| 항목 | 값 |
|---|---|
| anchor | `below_text` |
| 텍스트 영역과 간격 | 30px |
| 좌우 패딩 | 0 |
| fit | `contain_focal` — 영역 안에 비율 유지 contain |
| 정렬 | 콘텐츠 bounds(알파 채널 기준)의 visual center를 영역 중앙에 |

**영역 계산:**
- 세로: `(텍스트 영역 끝 + 30) ~ (캔버스 h - margins.bottom)`
- 가로: `margins.left ~ (preset.w - margins.right)` (= 472px)
- 텍스트 영역 끝은 로고/서브카피 유무에 따라 동적으로 계산됨

**수동 조정:** 사용자 드래그·슬라이더 보정값이 자동 배치 위에 추가로 적용. 학습 데이터 누적 가능.
