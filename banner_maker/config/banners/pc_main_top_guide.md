- **위치:** PC > 알바몬 메인 좌측

- **레이아웃:** 가로형 (horizontal)

## 레이아웃 구조

- **텍스트 영역:** 좌측 55%
- **이미지 영역:** 우측 45%
- **여백:** 좌 80 / 우 80 / 상 52 / 하 52 px
- **출처 라벨:** 16px, 우측 상단

### 이미지 박스

- 너비 비율: 40%
- 높이 비율: 95%
- 위치: 우측 중앙 (right_center)
- 상하 패딩: 8px
- 빌더 UI에서 imageScale·anchor로 미세 조정 가능

## 색상 규칙

- 페이지 배경: #FFFFFF (흰색)
- **필수:** 메인 페이지(흰 #FFFFFF)와 구분되는 배경색 사용

### 텍스트 색상 판단

- 방식: 배경 밝기(luminance) 기반 자동 판단
- 기준값: 0.5
- 어두운 배경 → 흰색 텍스트 (#FFFFFF)
- 밝은 배경 → 검정 텍스트 (#171717)
- 최대 텍스트 색상 수: 2가지

---

## 타겟 오디언스

- **대상:** 구직자, 사장님
- 알바몬 메인 방문자는 구직자(알바생)와 사장님(자영업자)을 모두 포함
- 광고 컨텍스트에 맞춰 두 청자 중 하나 또는 양쪽을 향하도록 카피·비주얼을 잡는다

---

## 카피 규정

- **구조:** 서브 1줄 + 메인 1줄
- **줄 간격:** 12px
- **메인 줄바꿈:** 불가

| 구분 | 최대 글자수 | 권장 글자수 |
|------|-------------|-------------|
| 메인 | 17자 | 13~17자 |
| 서브 | 14자 | 8~13자 |

### 카피 톤

- **서브:** 맥락·후크·인용·캠페인명
- **메인:** 행동 유도·핵심 메시지

---

## 캐릭터 정책

- **캐릭터:** 알바몬 마스코트 (MONI)
- **사용 방식:** 조건부 사용

### 사용 가능 조건

1. 카피 컨셉에 맞을 때 (의상·소품 변형)
2. 광고주 자산(제품·로고)을 캐릭터가 들고 있는 결합형
3. 알바몬 자체 캠페인 광고

### 금지

- 조건과 무관한 단순 장식 사용 지양

---

## 구조 변형

### 1. 서브(위) + 메인(아래) — 기본 구조

- 카피 컨셉이 "맥락 → 결론" 흐름일 때
- 사례: case_1, case_2, case_3, case_4, case_6

### 2. 메인(위) + 서브(아래)

- 메인 카피를 후크로 먼저 던지고, 서브로 부연 설명할 때
- 사례: case_5

## 제작 사례 (Few-shot References)

### Case 1 — 찐 알바생

- **광고주:** 알바몬 자체
- **타겟:** 구직자
- **배경:** 보라 단색
- **텍스트 색상:** #FFFFFF
- **서브 카피:** 이 친구 찐입니다
- **메인 카피:** 일 잘한 찐 알바생 추천하기
- **비주얼:** MONI 카드 + 거품 효과
- **캐릭터 근거:** 자사 캠페인 단독 사용
